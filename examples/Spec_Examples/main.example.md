## What This Game Is

DROP THE GLOVES is a two-player strategic hockey card game. Each player picks a PCHC team and plays cards across 3 periods. Each period's outcome is determined by **chain battles** through 5 goalie-hole locations, resolved using the Level Battle Rolls (LBR) system. Goals are scored when chains reach length 1 or higher and beat the goalie's save percentage. A chain of 1 is a Weak Shot (goalie gets +10% save bonus); chains of 2 and above improve shot quality progressively.

**Design Goals:**
- Quick game (~8 minutes)
- Deck = 15 skaters (9 F/C + 6 D) + 1 goalie + 1 coach
- 5 goalie-hole locations in a pyramid layout (1,2 top — 5 center — 3,4 bottom)
- Chain-based puck movement through pattern sequences
- 3 periods with 3 patterns each (6 chain runs per period)
- Face-up/face-down placement with boons and penalties
- 1989-90 O-Pee-Chee hockey card aesthetic with ice rink UI theme

## Current State (v1.0.0 — What's Working)

- **Vintage bitmap-font UI**: `ASSETS/IMAGES/UI/font.png` sliced by `TOOLS/slice_font.py` (68 glyphs A-Z, a-z, 0-9, -,.!?:). `SRC/UI/bitmap_font.py::BitmapFont` renders the sheet, with digit/symbol cap-height normalization + per-row baseline inference (descenders, commas, dots aligned). `BitmapFontWrapper` is a `pygame.font.Font` drop-in — `AssetLoader.load_font()` and `UIConfig.font()` route through it above size 16; below that falls through to TTF for stats/table legibility. Two-step ADD→MULT tint keeps anti-aliased edges color-clean.
- **Image-backed buttons**: `button5/10/15/20/25.png` ice-scratched panels picked by label length in `SRC/UI/components.py::Button` (incl. a `draw_image_button` classmethod for game_window's ad-hoc buttons). Scaled to requested height, label auto-sized, hover flash overlay.
- **Role-colored card outlines**: red/blue/green/yellow `*_outline.png` overlaid on hand cards and placed cards (selected = yellow, coach = green, defenseman = blue, forward = red). Applied to the hover blow-up preview too.
- **Animated card collage background**: 24-frame seamless sine-drift loop @ 12 fps (2-second loop), generated by `TOOLS/generate_collage_animation.py` with 280 cards on a 20×14 jittered grid with bleed edges and phase-offset per-card drift/rotation. Jamieson card pinned as a hero.
- **Web builds (pygbag)**: Two targets — desktop (`TOOLS/build_web.py`, 1600×1000 landscape) and mobile (`TOOLS/build_web_mobile.py`, 600×900 portrait). Both scripts now use hand-crafted HTML templates (`WEB_BUILD/index_desktop.html`, `WEB_BUILD/index_mobile.html`) instead of string-patching pygbag's generated output; only the bundle name and `BUILD_VERSION` timestamp are substituted per build. The mobile template includes a `MutationObserver` canvas enforcer — it watches for pygbag's own canvas style writes and immediately re-enforces the correct portrait dimensions, plus a `settle_and_resize()` function (double-rAF + staggered timeouts) that waits for the mobile browser's viewport to settle before the first enforce. Outputs `WEB_BUILD/hockey_game_web.zip` and `WEB_BUILD/hockey_game_web_mobile.zip`. OGG audio, cache-busting, portrait rotate-hint on mobile.
- **9 PCHC teams** with real 2025-26 NCHC-based rosters, stats-weighted LBR values (135 skaters + 9 goalies + 9 coaches, each with unique card art)
- **Team identities**: Ball-O-Devils, Felines, Mile-High, RedTurtles, Poodles, Fighting Scots, Red Cows, Habitants, Mutton Busters
- **LBR combat system** (Level Battle Rolls): percentage-based resolution — 50% (tie) → 60% → 75% → 85% → 95%; every battle rolls the dice including ties; puck sprites A-E
- **Unique player cards**: 153 named cards with custom letter sprites stamped on orange ribbon, per-team Y offsets, wide letter spacing
- **Animated PCHC logo puck**: follows the active pattern during resolution with glide animation, fade-in/fade-out, jiggle on goals; works in regulation and OT
- **Pattern visualization**: gradient paths (dark→transparent showing direction), square start marker, purple P4 color, only active pattern shown during resolution
- **Full game loop**: Menu → Team Select → Difficulty → Place P1 → Resolve P1 → Place P2 → Resolve P2 → Place P3 → Resolve P3 (+OT) → Game Over
- **Per-period resolution**: each period resolves immediately after both players finish placing it. Face-down card identities stay hidden across all periods — only the LBR at each resolved battle location is revealed, with full identities flipping only at end of game (desktop + mobile).
- **Comeback Roll ("The Grinder")**: skater-vs-skater battles lost by exactly 1 LBR gap get a second roll; if either roll wins, the underdog wins the location. Coach and goalie battles are unaffected. Raises gap-1 underdog win rate from ~40% to ~64%.
- **Opponent deck-stats tooltip**: hover / drag over any location to see the opponent's remaining A/B/C/D/E count at that location, based on cards you've seen. Public info — just saves the mental math.
- **3 AI difficulty levels**: Rookie (Jack Jamieson), Veteran (Chris Terrien), All-Star (Gretzky)
- **Two-phase bot placement**: bots choose face-up cards, then counter-place face-down based on opponent reveals
- **9 arena locations**: boon (3 face-up) / penalty (1 face-up) — themed per team (Devil's Furnace, Thin Air, etc.)
